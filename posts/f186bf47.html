目录与路径

目录的相关操作

  • cd (change directory, 变换目录)

  • pwd(显示目前所在的目录)

  • mkdir(建立新目录)

  • rmdir(删除空目录)

关于执行文件路径的变量: $PATH

PATH(一定是大写)这个变量的内容是由一堆目录所组成的，每个目录中间用冒号(:)来隔开， 每个目录是有『顺序』之分的

文件与目录管理

ls(文件与目录的检视)

  • ls [-aAdfFhilnrRSt] 文件名或目录名称

  • ls [–color={never,auto,always}] 文件名或目录名称

  • ls [–full-time] 文件名或目录名称

cp(复制文件或目录)

  • cp [-adfilprsu] 来源文件(source) 目标文件(destination)

rm(移除文件或目录)

  • rm [-fir] 文件或目录

mv (移动文件与目录，或更名)

  • mv [-fiu] source destination

文件内容查阅

cat(concatenate)

tac(反向显示)

nl (添加行号打印)

more(一页一页翻动)

less(一页一页翻动)

资料提取

head(取出前面几行)

tail(取出后面几行)

od(非纯文本文档)

touch(修改文件时间或建立文件)

文件与目录的默认与隐藏权限

umask(文件预设权限)

chatter(配置文件隐藏属性)

lsattr(显示文件隐藏属性)

Set UID

Set GID

Sticky Bit

file(查看文件类型)

指令与文件的查找

which(查找指令)

whereis(由一些特定的目录中查找文件文件名)

locate / updatedb

find
